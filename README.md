# Tạo Ảnh AI

Đây là bản phát hành extension `Tạo Ảnh AI` dành cho người dùng cài vào Google Chrome.

## Cài đặt nhanh

1. Tải file `TaoAnhAI.zip`.
2. Giải nén file ZIP.
3. Mở Chrome và truy cập `chrome://extensions`.
4. Bật `Developer mode`.
5. Bấm `Load unpacked`.
6. Chọn thư mục vừa giải nén có file `manifest.json`.

## Hướng dẫn sử dụng

Mở file `HUONG_DAN_SU_DUNG.html` để xem hướng dẫn đầy đủ, dễ đọc trên trình duyệt.

Nếu cần bản Markdown, xem `HUONG_DAN_SU_DUNG.md`.
